 Estructura del proyecto
- catalogo.html : Pagina principal (En este caso) y Catálogo completo
-  src/ : Contiene el resto del proyecto
-  data/ : contiene la semilla .js
- helpers/ : Contiene los helpers .js
- pages/ : contiene todos los html
- recursos/ : Contiene logos e imagenes
- Styles/ : contiene el archivo .css



Requerimientos:

El sistema debe permitir navegar entre las
categorías: Todo, Tecnología, Gestión e
Innovación a través de botones visibles
en todas las páginas. La página principal
mostrará todos los cursos disponibles.
Cada categoría tendrá su propia página
mostrando sólo los cursos relacionados.
Los cursos se mostrarán en tarjetas con
imagen, título, descripción, categoría y un
video explicativo.

Cada tarjeta debe contener: Imagen
representativa del curso. Nombre del
curso. Categoría a la que pertenece.
Descripción breve. Video con controles
para reproducir contenido.
